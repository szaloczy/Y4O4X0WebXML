Miaz az XML:

Az XML az angol Extensible Markup Language, vagyis „kiterjeszthető jelölőnyelv” rövidítése. Az XML lehetővé teszi az egyéni információs formátumok létrehozását, illetve elektronikus megosztását. Az XML-ben (és az ezen alapuló nyelveken) írt fájlokat még az olyan szoftverek is képesek olvasni és módosítani, amelyek nem ismerik az XML nyelvet.

Az XML sokban hasonlít a HTML-re (Hypertext Markup Language), azonban a W3C (World Wide Web Consortium) inkább az XML használatát javasolja. Mindkét nyelv jelölő szimbólumokat használ egy weboldal vagy fájl tartalmainak leírására.

A HTML kód azt határozza meg, hogy a weboldal tartalmai (általában a szövegek és a képek) hogyan jelenjenek meg és mit lehessen csinálni velük. Ezzel szemben az XML adatok „önmagukat írják le”, vagy „határozzák meg”, ami azt jelenti, hogy hogy az adatok szerkezete az adatokkal együtt kerül tárolásra, tehát amikor ezek beérkeznek, nincs szükség előzetes szerkezetre a tárolásukhoz – azokat az XML-en belül dinamikusan lehet értelmezni.

Az XML formátumot bárki, vagy bármilyen csoport, vállalat, szervezet, stb. használhatja, aki, vagy ami szeretné információit egy egységes módon megosztani. Az XML az SGML (Standard Generalized Markup Language, azaz „szabványos általános jelölőnyelv”) egyszerűsített, könnyebben használható változata.
